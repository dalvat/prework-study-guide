HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section element are semantic elements that define a section in a document that contains thematic content.
  • HTML uses the id attribute as a way to identify elements and select them. This will make more sense once we start using CSS and JavaScript.

Git

  • git status: checks what branch we are currently on
  • git checkout main: navigates back to the main branch
  • git add -A: commits all files in the working branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: checks the local main branch and github version are the same

Commandline

  • ls: lists the folders in the current directory
  • cd directoryname: navigates to a directory
  • cd ..:navigates up one level/folder
  • touch directoryname: creates a new folder/directory
